# Welcome to Tzmicha Mehasafa landing page project

## Project info

**URL**: https://tzmicha-mehasapa.com

## What it's about?

A modern, responsive landing page designed to help families discover financial stability and independence through professional coaching by **Liran Aharon** — a financial mentor.

The goal of the site is to collect leads for a **paid financial growth training**, custom made to each customer.
Visitors are encouraged to watch a short YouTube video and free video training on financial literacy, pensions, savings, and debt recovery.

## 🧭 What’s on the Page?

- ✍️ **Personal Story** – Liran’s background and real-life success story helping a families succeed financially.
- 🎥 **Video Section** – A YouTube video training on pensions, savings, and debt recovery.
- 📩 **Leads Capture Form** – Collects user details for more guidance.
- 💬 **Motivational Content** – Clear, engaging Hebrew text to inspire trust and action.

## 🚀 Features

- Fully responsive design across mobile and desktop
- YouTube Shorts video integration
- Animated call-to-action sections
- Contact form for leads collection
- Clean and animated UI using Tailwind CSS
- Integration to social media accounts
- Integration to Whatsapp chat
- Custom made business content

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
